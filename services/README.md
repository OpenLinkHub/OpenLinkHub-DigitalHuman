# Services

Backend and AI service implementations live here.

Recommended grouping:

- `asr`: Speech recognition providers and ASR gateways.
- `tts`: Text-to-speech providers.
- `conversation`: Realtime dialogue orchestration and session state.
- `rag`: Knowledge ingestion and retrieval.
- `avatar`: Digital human rendering and streaming.
- `tools`: OpenLinkHub business tool adapters.
- `gateway`: Frontend-facing BFF or API gateway.
